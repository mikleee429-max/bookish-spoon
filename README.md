# Fuzzy Engine 🎮

## Описание

Полнофункциональное веб-приложение для игр с системой профилей, рейтингов, магазина и турниров. Разработан для демонстрации простых мини-игр и интерфейса.

## Установка

1. Распакуйте архив
2. Откройте `index.html` в браузере
